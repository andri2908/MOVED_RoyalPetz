STORE KEY BUTTON
- Store the RSA Parameters to Container
- Container Name will follow the textbox value, default will be "POS_CONTAINER"

DELETE KEY BUTTON
- Delete the KEY that was stored inside the Container
- Container Name will follow the textbox value, default will be "POS_CONTAINER"

CREATE LICENSE FILE
- Create and Encrypt License File based on the RSA Parameters stored in Container
- License File value are consist of ProcessorID + HDD S/N
- Default filename for license file that's hardcoded in the Royal Petz Code is "license.lic"

CHECK LICENSE FILE
- To read and Decrypt the file based on the RSA Parameters stored in Container
